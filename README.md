# Описание проекта

Данный репозиторий содержит пример приложения на C++, которое использует OpenGL для рендеринга графики и WinAPI для создания окна. Проект демонстрирует базовые принципы работы с графикой в Windows, включая:

- Инициализацию контекста OpenGL.
- Создание и управление окном через WinAPI.
- Рендеринг простых геометрических фигур.
- Структурированную сборку проекта с помощью CMake.

## Структура проекта

- `src/` — исходный код приложения.
  - `src/window/` — обработка оконных сообщений и создание окна.
- `res/` — ресурсы проекта.
  - `res/main.ico` — иконка окна.
  - `res/winmain.rc` — скрипт ресурсов Windows.
- `CMakeLists.txt` — файл сборки CMake, обеспечивающий кроссплатформенную сборку.
- Скрипты сборки:
  - `build_debug.bat` — сборка в режиме отладки.
  - `build_release.bat` — сборка в релизном режиме.
  - `clean.bat` — очистка каталога сборки.

## Как собрать и запустить

1. Откройте командную строку в корне проекта.
2. Для сборки в режиме отладки выполните:
   ```bat
   build_debug.bat
   ```
3. Для сборки в релизном режиме выполните:
   ```bat
   build_release.bat
   ```
4. После сборки запустите соответствующий исполняемый файл в каталоге `bin/` (или аналогичном, в зависимости от настроек CMake).
