# OGLE20

Проект 3D-приложения/движка, разработанного на C++ с использованием OpenGL.

## Статус проекта

Этот проект находится на ранней стадии разработки. Структура и функциональность могут изменяться.

## Для ИИ-агентов

Этот раздел содержит рекомендации для ИИ-ассистентов по эффективной навигации и работе с проектом.

- **Тип проекта:** C++ приложение с использованием OpenGL.
- **Система сборки:** CMake. Для удобства используйте `build.bat` или стандартные команды CMake.
- **Основная точка входа:** `src/main.cpp`.
- **Ядро рендеринга:** Расположено в `src/opengl/`. Ключевые файлы: `OpenGLRenderer.h` и `DomoScene.h`.
- **Управление сценой и миром:** `src/world/World.h` и `src/models/`.
- **Оконная система:** Код для работы с окнами находится в `src/ui/`. В настоящее время используется Win32 (`Win32Window.h`).
- **Зависимости:** Управляются через CMake FetchContent и перечислены в `CMakeLists.txt`. Включают Assimp, GLM, nlohmann_json и OpenMesh.
- **Ресурсы:** 3D-модели, текстуры и другие ресурсы хранятся в каталоге `assets/`.
- **Бинарные файлы:** Скомпилированные исполняемые файлы и необходимые DLL находятся в каталоге `bin/`.

## Структура каталогов

```
e:\my_proj\OGLE20
├───assets/         # 3D-модели, текстуры и другие ресурсы
├───bin/            # Скомпилированные бинарные файлы и зависимости времени выполнения
├───build/          # Файлы сборки, сгенерированные CMake
├───src/            # Исходный код
│   ├───models/     # Модели данных и структуры для 3D-объектов
│   ├───opengl/     # Реализация рендеринга на OpenGL
│   ├───ui/         # Код для работы с окнами для конкретной платформы
│   ├───world/      # Управление миром/сценой
│   ├───App.h/.cpp  # Основной класс приложения
│   └───main.cpp    # Точка входа в приложение
├───mdocs/          # Дополнительная документация
├───.github/        # Файлы для GitHub (например, CI/CD, шаблоны)
├───build.bat       # Основной скрипт сборки для Windows
└───CMakeLists.txt  # Корневой файл конфигурации CMake
```

## Зависимости

Проект использует следующие внешние библиотеки, которые автоматически загружаются с помощью CMake:
- **assimp:** для импорта 3D-моделей.
- **glm:** математическая библиотека для OpenGL.
- **nlohmann/json:** для парсинга JSON.
- **OpenMesh:** для обработки треугольных сеток.

## Как собрать проект

### Требования
- Компилятор C++ (например, MSVC, GCC, Clang)
- CMake (рекомендуется версия 3.15 или выше)

### Шаги сборки (Windows)

1.  Откройте командную строку или терминал.
2.  Перейдите в корневой каталог проекта.
3.  Запустите скрипт сборки:
    ```sh
    .\build.bat
    ```
4.  Этот скрипт настроит проект с помощью CMake и выполнит сборку. Файлы решения и проекта будут созданы в каталоге `build/`.

### Ручная сборка с помощью CMake

В качестве альтернативы можно использовать стандартные команды CMake:
```sh
mkdir build
cd build
cmake ..
cmake --build .
```

## Как запустить

После успешной сборки исполняемый файл `OGLE3D.exe` и его зависимости будут находиться в каталоге `bin/`.

1.  Убедитесь, что ваш текущий каталог — это корень проекта (`e:\my_proj\OGLE20`).
2.  Запустите приложение из каталога `bin`:
    ```sh
    .\bin\OGLE3D.exe
    ```
